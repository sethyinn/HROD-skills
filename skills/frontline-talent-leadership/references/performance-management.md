\# Performance Management



Performance management ensures fair evaluation.



---



\# Performance Dimensions



| Dimension | Example          |

| --------- | ---------------- |

| Results   | targets achieved |

| Behaviors | teamwork         |

| Skills    | expertise        |

| Growth    | improvement      |



---



\# Performance Review Process



1 employee self-review

2 manager evaluation

3 calibration discussion

4 final rating



---



\# Performance Ratings



| Rating            | Meaning              |

| ----------------- | -------------------- |

| Outstanding       | exceeds expectations |

| Strong            | meets expectations   |

| Needs Improvement | below expectations   |
